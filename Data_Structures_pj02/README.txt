Bradley Pierson - head.cpp and tail.cpp
To run head, type $./head -n [number] [file...]. if [number] is not specified,
the defualt value is 10. If [file] is not specified or specified as "-", then
standard input is read up to 4096 bytes. Program prints the first n number
of lines of file/standard input.

///////////////////////////////////////////////////////////////////////////////
IN ORDER TO END READING FROM STANDARD INPUT, THE USER MUST TYPE "end" and
hit enter or manually input "end\n"
///////////////////////////////////////////////////////////////////////////////

To run tail, tyype $./tail -n [number] [file]. If [number] is not specified,
the defualt value is 10. If [file] is not specified or specified as "-", then
standard input is read up to 4096 bytes. Program prints the last n number
of lines of file/standard input.

///////////////////////////////////////////////////////////////////////////////
IN ORDER TO END READING FROM STANDARD INPUT, THE USER MUST TYPE "end" and
hit enter or manually input "end\n"
///////////////////////////////////////////////////////////////////////////////
